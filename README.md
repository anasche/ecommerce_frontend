# Frontend - Full-Stack Assessment

This is the frontend application for the Full-Stack Assessment Assignment.  
It is built using **Next.js 14** and connects to the Laravel backend API.

## Features

- Display list of products
- View product details
- Add products to cart (state-based)
- Place order through backend API
- Loading and error states
- Clean component structure

## Tech Stack

- Next.js 14
- React 18
- Axios (for API calls)
- Node.js 18+
